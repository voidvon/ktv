# 快速开始

这份入口页只保留第一次使用最需要的信息。

## 先记住三件事

- Android 是当前最完整、最适合优先体验的平台。
- 第一次启动后，先配置歌库来源，再回到首页点歌。
- 常用流程就是搜索歌曲、加入已点、播放时切换原唱或伴唱。

## 支持平台

| 平台 | 当前发布形态 | 建议用途 |
| --- | --- | --- |
| Android | APK | 主力验证平台，功能最完整 |
| macOS | 桌面包 | 桌面和大屏测试 |
| Windows x64 | 桌面包 | 桌面测试与基础验证 |
| iOS | unsigned IPA | 真机侧载与播放器链路测试 |
