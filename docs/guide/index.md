# 快速开始

这份文档面向第一次接触麦麦KTV的测试用户、协作者和内部使用者。你可以先从这里理解这个软件能做什么、当前适合怎么安装，以及第一次启动后应该如何准备歌库。

## 适合谁

- 想在电视、投屏设备或桌面端快速搭建一套家庭 KTV 点歌环境的人
- 想管理本地歌库、继续扩展云端歌曲来源的人
- 需要验证 Android、macOS、Windows x64 或 iOS 测试包的人

## 支持平台

| 平台 | 当前发布形态 | 建议用途 |
| --- | --- | --- |
| Android | APK | 主力验证平台，功能最完整 |
| macOS | 桌面包 | 桌面和大屏测试 |
| Windows x64 | 桌面包 | 桌面测试与基础验证 |
| iOS | unsigned IPA | 真机侧载与播放器链路测试 |

## 首次使用建议

1. 先从 GitHub Releases 下载对应平台的测试包。
2. 安装后进入设置页，接入本地歌曲目录，或登录并配置百度网盘歌曲源。
3. 回到首页，从“按歌名点歌”或“按歌星点歌”进入歌库。
4. 用中文、拼音首字母或歌手名搜索，把歌曲加入已点列表。
5. 播放过程中按需切歌、重唱，或切换原唱与伴唱。

## 歌库来源

目前软件支持以下歌库来源：

- 本地目录歌曲
- 百度网盘歌曲源

如果你需要了解更细的云盘接入方向，可以继续看 [云盘数据源可复用流程](/cloud_drive_source_reusable_flow) 和 [百度网盘接入准备](/baidu_pan_data_source_guide)。
