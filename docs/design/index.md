# UI 设计入口

这里收拢的是当前 KTV 大屏界面相关的设计资料，包括布局结构、设计 Token、状态矩阵、文案与资源说明。

## 推荐阅读顺序

1. [UI 复刻规格文档](/ktv_ui/ui_rebuild_spec)
2. [KTV UI Design Tokens](/ktv_ui/ui_design_tokens)
3. [KTV UI 结构线框说明](/ktv_ui/ui_layout_wireframe)
4. [KTV UI 状态矩阵](/ktv_ui/ui_state_matrix)
5. [KTV UI 文案清单](/ktv_ui/ui_copywriting_inventory)
6. [KTV UI 资源说明](/ktv_ui/ui_asset_notes)
7. [UI 组件检查表](/ktv_ui/ui_component_checklist)

## 适用场景

- 重做首页或点歌工作区
- 调整横屏与紧凑布局
- 排查播放层、设置弹窗和状态卡的视觉行为
- 对齐文案、图标、背景装饰和资源复刻方式

## 使用建议

先看规格，再看状态矩阵，最后对照文案与资源说明落实现。这样可以减少只改表面样式但漏掉交互状态和信息语义的问题。
